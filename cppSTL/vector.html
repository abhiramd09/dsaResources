Introduction

  • It is a dynamic array , has ability to resize itself.
  • Inserting an element at the end takes differential time, removing last element takes only constant time.
  • Inserting and erasing at the beginning or middle is linear in time.

Declaration and Insertion

  • A vector is declared as vector<'datatype'>v;
  • v.push_back('element_of_datatype'), adding element to vector v, element is added at the end of the vector.
  • vector<vector<int>>V, declaration of a 2D vector.
  • vector<vector<int>>V(n, vector<int>(m)), creating a vector with n rows and m columns.

Properties

  • v.size(), returns the number of elements in vector.
  • v.capacity(), returns the size of storage space currently allocated, expressed as number of elements.
  • v.max_size(), returns the max number of elements a vector can hold.
  • v.resize(n), resizes container so it contains n elements.
  • v.empty(), returns wheather the container is empty or not.
  • v.shrint_to_fit(), reduces the capacity of container to fit its size and destroy all elements beyond capacity.

Element Access

  • v[2], v.at(2), returns reference to the element at position 2.
  • v.front(), returns reference to the first element in vector.
  • v.back(), returns reference to the last element in vector.
  • v.data(), returns a direct pointer to memory array used internally by the vector to store its owned elements.

Modifiers

  • v.assign('index', 'newValue'), assigning a new value at index replacing the old ones.
  • v.pop_back(), removing last element.
  • v.insert(v.begin(), 5), inserts 5 at the beginning.
  • v.erase(v.begin()), removes first element.
  • v.emplace(v.begin(), 5), extends container by inserting new element.
  • v.emplace_back(20), inserts 20 at the end.
  • v.clear(), erases the vector.
  • v1.swap(v2), swaps v1 and v2(same type, but size may differ).
  • replace(v.begin(), v.end(), old_val, new_val), replacing old value with new value.
  • reverse(v.begin(), v.end()), reverses the order of elements in vector.

Iterations

  • We need to add the #include<iterator> to use the iterator.

  • for(auto i = v.begin() ; i!= v.end() ; ++i)

    This loop iterates the elements of vector from start.

  • for(auto i= v.rbegin(); i!= v.rend() ; i++)

    This loop iterates the elements from last.

  • v.rbegin(), returns a reverse iterator pointing to the last element in the vector(moves from last to first).
  • v.rend(), returns a reverse iterator pointing to theoritical element preceeding first element in vector.
  • v.begin(), returns iterator pointing first element in vector.
  • v.end(), returns an iterator pointing to the theoritical element that follows last element in the vector.
  • v.cbegin(), returns a constant iterator pointing to first element in vector.
  • v.cend(), returns a constant iterator pointing to the theoriticaal element that follows last element in vector.
  • v.crbegin(), returns a constant reverse iterator pointing to last element in vector (moves from last to first).
  • v.crend(), returns a constant reverse iterator pointing to theoritical element preceeding the first element in vector.

Sorting

  • sort(v.begin(), v.end()), sorts the elements in ascending order .
  • sort(v.begin(), v.end(), greater<int>()), sorts the elements in desending order .

Finding an element in vector

  • Find function returns first element in the range that compares equal to the value, if no element matches, the function returns last.
  • Let there be a vector v and vector iterator it (initialised as vector<int> : : iterator it)
  • it = std : : find(v.begin(), v.end(), 'element_to_be_checked')
  • if it is not equal to v.end(), then element is located at it-v.begin(), else element is not found.

Search

  • Finding out if a vector is subsequence of another vector.
  • Let there be two vectors v1 and v2 and one iterator it.
  • it = std : : search(v1.begin(), v1.end(), v2.begin(), v2.end());
  • if it is not equal to v1.end(), then vector 2 is present at index it-v1.begin(), else vector 2 is not present.
