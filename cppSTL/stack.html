Introduction

  • Stack is a Last In First Out (LIFO) type of implementation.
  • Elements are added and removed at the same end i.e, top.

Declaration and Properties

  • stack<'datatype'>s; this is the declaration of stack.
  • s.push('element') adds the element at the top of the stack.
  • s.pop('element') deletes the topmost element of the stack.
  • s.empty() returns a boolean value weather the stack is empty.
  • s.size() returns the size of the satck.
  • s.top() returns the reference to the topmost element of the stack.
  • s1.swap(s2) swaps tne contents of one stack with another.
