Introduction

  • Similar to queue but first element of the queue is greatest of all elements in the queue.
  • Elements are in non-increasing order, each element of the queue has a fixed priority.
  • A priorityQueue is also called a max-heap.

Declaration and Properties

  • priority_queue< 'datatype'>pq; this is how a priority queue is declared.
  • pq.push('element') adds an element to the queue, not at the end.
  • pq.pop() deletes the first element of the queue.
  • pq.top() returns a reference to the topmost element of the queue.
  • pq.size() returns the size of the queue.
  • pq.empty() returns wheather the queue is empty.
  • pq1.swap(pq2) swaps the content of two queues of same type.

Min-Heap

  • We can create a min-heap for the priority queue.
  • By default c++ creates a max heap for priority queue.
  • The declaration for min-heap is priority_queue<int, vector<int>, greater<int>>minHeap; the elememts are in increasind order.
