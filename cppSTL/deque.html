Introduction

  • Similar to vectors but efficient in case of insertion and deletion.
  • Double ended queue are special case of queues where insertion and deletion are possible at both ends.

Declaration and Properties

  • A deque is declared as deque<'datatype'> d;
  • d.push_back('element') pushes element into deque from back.
  • d.push_front('element') pushes element into deque from front.
  • d.size() returns the size of number of elements in deque.
  • d.front() referes to the first element of deque.
  • d.back() referes to the last element of deque.
  • d.pop_front() pops the element from front.
  • d.pop_back() pops the element from back.
  • d.resize() chamges the size deque.
  • d.clear() deque becomes empty.
  • d.empty() checks if the deque is empty or not.
  • d.at(i) reference to the element at the given position
  • d1.swap(d2) swaps the elements of deque.

Iteration

  • First we need to declare an iterator deque<'datatype'> :: iterator it = dq.begin(); and itertate this over the deque.
  • dq.insert(it, 10) inserting 10 at position it.
  • dq.insert(it, 2, 0) inserting 0's 2 times at position it.
  • dq.insert(it, v.begin(), v.end()) inserting a vector vector<int>v to the deque.
  • dq.erase(it) removes the elelemt from particular posistion.
