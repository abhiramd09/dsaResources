Introduction

  • Queue is a First In First Out type of implementation.
  • Elements are inserted at the back and are deleted from front.

Declaration and Properties

  • Queue is declared as queue<'datatype'>q;
  • q.push('element') adds an element at the end of the queue.
  • q.front() returns a reference to the first element of the queue.
  • q.back() returns a reference to the last element of the queue.
  • q.pop() deletes the first element of the queue.
  • q.size() returns the size of the queue.
  • q.empty() returns a boolean value weather the queue is empty.
  • q1.swap(q2) exchanges the content of two queues, but queues must be of same data type, although size may differ.
