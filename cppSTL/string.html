Introduction

  • Stirng is different from the character array which we see in C.

Declaration

  • Stirng s;
  • To take string input from console we write : getline(cin, s).

Insertion and Deletion

  • To insert elements at the end of the string: s.push_back('char_you_want_to_insert');
  • To delete last character from string: s.pop_back( );
  • s.erase( ), erases the whole string.
  • s.erase(i, k), erases k characters from index i.

Properties

  • Length of the string : s.length( ).
  • s.capacity( ), returns the capacity allocated to the string which can be equal or more than the size of string.
  • s.shrink_to_fit( ), decreases the capacity and makes it equal to minimum capacity of string.
  • s1.swap(s2), swaps one string with another.
  • s1.copy('target character array', 'starting position in the string to start copying', 'length to be copied'), copies a portion of string to a character array.

Reversing

  • reverse(s.begin(), s.end()), reverses the original string.
  • String reverse_string = string(s.rbegin(), s.rend()), copies the reverse of a string to a new string.

Merging two strings

  • s1.append(s2), appends s2 at the end of s1 (s1s2 - resultant string).
  • Simply we can merge them using '+' operator, String newMergedString = s1+s2;

Finding a substring

  • s1.find(s2), it checks if s2 is a substring of s1. If there is a substring, it returns the starting index of the string else it returns -1.
  • s.substr(i, j-i+1), it returns the substring from index i to j.

Conversions

  • isupper('char'), returns a boolean value true if the arguement contains any uppercase values.
  • islpper('char'), returns a boolean value true if the arguement contains any lowercase values.
  • toupper('char'), converts a character to uppercase.
  • tolpper('char'), converts a character to lowercase.

String to int Conversion

  • stoi(s), converts a string arguement to int from.
  • int(char), returns the ascii value of the charcater.

  • Using string streams we can convert a string to interger.

    Let s be the string of numbers, we write stringstream g(s);

    After that int x = 0; and g>>x;

    x is the integer form of string s.

  • Suppose s1 and s2 are two integer string we can compare both by s1.compare(s2) return >0 if s1>s2, return <0 if s1<s2

int to String Conversion

  • to_string(a), converts an integer to string and returns the string value.
  • char(int), returns the character with ascii value int.
