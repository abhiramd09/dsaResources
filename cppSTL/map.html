Introduction

  • Stores elements in a mapped function.
  • Each element has a key value and a mapped value.
  • No two mapped values can have the same type of key value.
  • There are four types of maps
    1. map.
    2. unordered_map.
    3. multimap.
    4. unordered_multimap.

Map

Declaration

  • map< 'datatype1, datatype2'> m; this is a declaration of an empty map.
  • m.insert(pair<int, int>(1, 2)); this is way to insert elements into map, assuming the two datatypes are integrs.
  • map<int, int>m2(m1.begin(), m1.end()); assigning elements from map m1 to map m2.

Iteration

  • First we need to declare an map iterator as map<int, int> :: iterator itr;

    for(itr = m.begin(); itr!=m.end(); itr++)

    { cout<<itr->first <<itr->second; } this prints the key and value of that particular key.

  • We can also iterate a map with auto iterator without actually declaring the map iterator.

    for(auto i : m)

    cout<<i.first<<i.second;

Properties

  • m.erase(m.begin(), m.find(3)); removes all the elements upto element with key = 3 in m.
  • m.erase(4) removes all the elements with key =4.
  • m.size() returns the number of elements in map.
  • m.empty() returns whether the map is empty.
  • m.clear() removes all the elements from map.
  • m.at(3) returns the associated or mapped element with key value 3.
  • m1.swap(m2) or swap(m1, m2) swaps the contents of two maps.
  • m.count(n) returns the number to element n in map.

Multimap

Introduction

  • Similar to map with addition that multiple elements can have the same key.
  • Not required that key value and mapped value to be unique.
  • Multimap keeps all the keys in sorted order always.
